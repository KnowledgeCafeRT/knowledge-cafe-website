# Pecita Font

To use the Pecita font correctly:

1. Download the Pecita font from:
   - Font Squirrel: https://www.fontsquirrel.com/fonts/pecita
   - DaFont: https://www.dafont.com/pecita.font
2. Place the font file(s) in this directory with one of the following names:
   - `Pecita.woff2` (preferred)
   - `Pecita.woff` (optional)
   - `Pecita.otf` (optional)

The CSS will automatically load the font once the file is in place.
